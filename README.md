# Kubernetes Administration Training Repository

© 2026 QA

## Overview

This repository contains **training materials, example manifests, lab artefacts, and supporting resources** used in Kubernetes administration and cloud-native training courses.

The content is designed to support **hands-on learning and instructor-led demonstrations** covering topics such as:

- Kubernetes architecture
- Pods, ReplicaSets and Deployments
- Services and Networking
- Storage and Volumes
- ConfigMaps and Secrets
- Helm
- RBAC and Kubernetes Security
- Policy as Code (Kyverno)
- Troubleshooting and Cluster Operations

The materials are intended to accompany structured training sessions and guided laboratory exercises.

---

## ⚠️ Training Use Only

This repository is provided **for training and educational purposes only**.

The manifests, scripts, and examples contained within this repository:

- Are designed for **demonstration purposes**
- May include **simplified implementations**
- May omit **security, resilience, or operational best practices**
- May intentionally contain **incomplete or incorrect configurations** as part of practical exercises

These examples **must not be used directly in production environments**.

## Intended Audience

This repository supports:

- Delegates attending instructor-led training courses
- Students completing guided laboratory exercises
- Instructors delivering demonstrations

Participants are expected to have:

- Basic Linux command-line skills
- Familiarity with containers and container images
- A general understanding of cloud computing concepts

Some modules may assume prior exposure to Docker or Kubernetes fundamentals.
